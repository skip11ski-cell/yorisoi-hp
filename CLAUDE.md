# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## プロジェクト概要

チャットボット付きのパーソナルホームページ。現在は `index.html` 単体のフロントエンドのみ構成。

## ルール

- **返答は日本語で行う**
- **コードのコメントは日本語で書く**
- **デザインはモダンで洗練されたものにする**（ダークテーマ・グラデーション・アニメーションを基調とした現在のスタイルを維持する）
- **レスポンシブ対応を必ず行う**（スマートフォンでも崩れないよう `clamp()` / `flex-wrap` / メディアクエリを使うこと）

## アーキテクチャ

現在は `index.html` 1 ファイルにすべてを記述（HTML・CSS・JS インライン）。

チャットボットを追加する際は以下の方針を想定：
- UI：`index.html` にチャットウィジェットを追加
- API 連携：Claude API（`@anthropic-ai/sdk`）または他の LLM API をバックエンドに配置
- バックエンドが必要になった場合は `server/` ディレクトリを作成して分離する

## デザインシステム

CSS 変数（`:root`）でトークンを管理している：

| 変数 | 用途 |
|---|---|
| `--bg` | ページ背景（ほぼ黒） |
| `--surface` | カード・チップの背景 |
| `--border` | ボーダー色 |
| `--accent` | メインアクセント（紫系 `#7c3aed`） |
| `--accent2` | サブアクセント（シアン系 `#06b6d4`） |

新しい色を追加する場合は必ずこの変数リストに追加すること。

## 背景レイヤー構成

`body::before` と `body::after` の 2 つの疑似要素で大理石柄を表現している：

- `::before` — CSS グラデーションによる斜めベイン＋放射グラデーション
- `::after` — SVG `feTurbulence` フィルターによる細かいひび割れ質感（`mix-blend-mode: screen`）

背景を変更する際はこの 2 層の重なりを壊さないよう注意すること。
